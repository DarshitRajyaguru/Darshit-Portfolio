Standby24

Website: standby24.co.uk

Standby24 UK is a professional healthcare staffing agency providing job opportunities and workforce solutions across the UK. This website was fully designed and developed from scratch, following a performance-first approach and business-aligned UX goals.

I created a unique, custom-designed **header and footer layout** that reflects the brand’s professionalism while enhancing usability and accessibility. The entire website was built using **Elementor** and extended with **Ultimate Add-ons** and other relevant widgets to ensure flexibility and responsiveness across all devices and screen sizes.

The site structure is powered by **Custom Post Types (CPTs)** and **Advanced Custom Fields (ACF)** to allow administrators to manage job listings, testimonials, and dynamic sections with ease. Each section was crafted with reusable and scalable logic for content control and layout consistency.

I integrated **Contact Form 7** across various touchpoints for candidate and client inquiries, and configured **SMTP email delivery via Microsoft Outlook** to ensure secure, reliable mail delivery. In addition, I implemented an **Advanced Database logging system** to track and store every form submission directly within the WordPress dashboard — making data export and review seamless for the client.

  • ✅ Fully custom-built design using Elementor and Ultimate Add-ons
  • ✅ Branded, responsive header & footer developed from scratch
  • ✅ CPT + ACF integration for dynamic job listings and sections
  • ✅ Contact Form 7 setup with Microsoft Outlook SMTP mailer
  • ✅ Advanced DB record keeping for form entry logging & exports
  • ✅ Optimized for mobile, tablet, and all modern browsers

OneCall24 Healthcare

Website: onecall24healthcare.co.uk

This project was all about building a tailored digital recruitment experience from the ground up — one that supports both candidates and administrators in the fast-paced healthcare staffing industry.

I began with a custom design approach using Elementor Pro and Ultimate Add-ons to create a clean, professional interface, with strong calls-to-action and mobile-friendly layouts. The goal: make job search and registration seamless on any device.

Aura Staffing

Website: imsnhance.in/aurastaffing

Aura Staffing is a healthcare recruitment platform crafted to connect top-tier professionals with healthcare opportunities across the country. The website was developed to deliver both high performance and flexible content management for the client’s evolving business needs.

The site was built using Elementor Pro, alongside Premium Addons and Ultimate Addons, to create modular, animated layouts. The interface was designed to be fully responsive and visually intuitive across all screen sizes.

For form management, I integrated both Contact Form 7 and Ninja Forms to serve different workflows — including inquiry handling, candidate submissions, and multi-step interactions. These were styled to match the brand and embedded across key sections to increase conversions.

To bypass the limitations of the parent theme, I created several custom templates using a child theme approach. This allowed me to control layout rendering, override default behaviors, and build in page-specific structures that Elementor alone couldn’t fully handle. The result is a stable, scalable, and easily maintainable website that performs well across devices.

  • ✅ Built with Elementor Pro + Premium & Ultimate Addons
  • ✅ Contact Form 7 and Ninja Forms for multiple use cases
  • ✅ Custom page templates for layout precision
  • ✅ Child theme setup for safe code and layout overrides
  • ✅ Fully responsive and optimized for all screens
  • ✅ Designed for clarity, accessibility, and performance
